# QA Automation Assignment

End-to-end test automation framework for the [Avtoikonom](https://dev.admin.avtoikonom.com) admin panel,
covering authentication and the Partners management flow (create, update, validate).

Built with **Playwright** and **TypeScript**.

> Work in progress — full documentation is added in a later iteration.

## Requirements

- Node.js `>=20` (see [`.nvmrc`](./.nvmrc))
